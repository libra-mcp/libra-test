# Rule: Interaction Patterns

## What to check
- Include clear loading feedback (skeletons/progress).
- State transitions should preserve context and remain interruptible.
- Feedback patterns (hover/tap/ripple/toast) should be subtle and not block primary actions.
- Gesture interactions (drag/swipe) should include clear thresholds and cancellation paths.
