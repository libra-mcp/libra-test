# Rule: Understandable (WCAG Principle 3)

## What to check
- Page and language metadata are set correctly (`lang` and language-of-parts).
- Behavior is predictable (no unexpected context change on focus/input).
- Inputs provide labels, instructions, and actionable error messaging.
- Critical forms provide error prevention/review flows where required.
