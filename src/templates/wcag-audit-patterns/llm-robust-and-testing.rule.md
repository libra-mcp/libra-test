# Rule: Robust and Testing (WCAG Principle 4)

## What to check
- Custom widgets expose valid name/role/value semantics and state updates.
- Status messages use appropriate live-region semantics.
- Combine automated checks (axe/pa11y/lighthouse) with manual keyboard/screen-reader testing.
- Provide concrete remediation snippets for each reported violation.
