# Rule: Perceivable (WCAG Principle 1)

## What to check
- Non-text content has correct alternatives (`alt`, decorative null alt, labels for meaningful icons).
- Media has transcript/caption/description as required.
- Semantic structure conveys relationships (headings, lists, tables, labels, landmarks).
- Color is not sole signal; contrast and reflow/text spacing requirements are met.
