# Rule: Design-System Workflow

## What to check
- Start with design-system generation before implementation details.
- Prefer stack-aware guidance (default `html-tailwind` when unspecified).
- Use page-level overrides only when intentionally diverging from master system rules.
- Before delivery, verify checklist: interaction feedback, dark/light contrast, responsive breakpoints, and accessibility essentials.
