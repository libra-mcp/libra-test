# Rule: Layout, Typography, and Performance (high)

## What to check
- Responsive layout avoids horizontal overflow and follows a consistent spacing system.
- Body text remains readable on mobile (minimum 16px equivalent).
- Reserve async space to prevent content jumps.
- Prefer transform/opacity animation and respect reduced-motion.
- Keep typography and color usage consistent with system-level tokens.
