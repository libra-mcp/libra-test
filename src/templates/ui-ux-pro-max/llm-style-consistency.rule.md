# Rule: Style and Visual Consistency (medium)

## What to check
- Chosen visual style matches product type and remains consistent across screens.
- Use SVG icon systems; avoid emoji icons for production UI.
- Maintain predictable z-index scale and consistent icon sizing.
- Hover/focus effects should not cause layout shift.
